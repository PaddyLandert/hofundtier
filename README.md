# Hof-, Pferde- & Kleintierbetreuung Landert

Dies ist die Website für **Hof-, Pferde- & Kleintierbetreuung Landert** im Zürcher Unterland.

Die Website informiert über das Angebot von **Yvonne Landert** in den Bereichen:

* Pferdebetreuung
* Kleintierbetreuung
* Stallhilfe
* Ferienbetreuung
* Unterstützung rund um Hof und Tiere

## Projektstatus

Aktuell handelt es sich um eine einfache statische Website auf Basis von:

* HTML
* CSS
* JavaScript

Die Seite ist für die Veröffentlichung über **GitHub Pages** vorgesehen.
